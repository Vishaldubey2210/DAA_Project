# 📘 Design and Analysis of Algorithms (DAA) - Complete Project & Lab Repository

Welcome to the comprehensive repository for the **Design and Analysis of Algorithms (DAA)** coursework. This repository combines the foundational algorithmic laboratory practices seamlessly alongside our major course project (*CrisisNet*).

---

## 🌟 1. Major Course Project: CrisisNet
**CrisisNet** is our flagship application built to analyze and manage disasters using modern algorithms. The project smoothly translates theoretical DAA techniques (such as Dijkstra's pathfinding and the Knapsack algorithm for resource allocation) into a functional web-based operational scenario.

### Included Project Documents
- 📄 **Project Report**: [`vishal prject report.pdf`](./vishal%20prject%20report.pdf) - Extensive details of the problem statement, system architecture, algorithm usage, and solution implementations. 
- 📊 **Presentation**: [`vishal ppt.pdf`](./vishal%20ppt.pdf) - Visual slides highlighting our project overview, methodologies, and concluding insights.

You can find the entire source code for the CrisisNet web application inside the [`/CrisisNet`](./CrisisNet) directory.
